# Rural-On-Demand-Self-Driving-Vehicle-project

## 1. 프로젝트 개요 및 동기
### - 문제 정의: 농촌 지역에서는 제한된 교통서비스로 인한 주민들의 이동권이 충분히 보장되지 못하고 있음
### - 문제 해결 방안: 농촌 주민들의 이동권 보장 및 삶의질 향상시키기 위한 자율주행차량 호출 시스템 개발
### - 목표: ROS2 Humble를 기반으로 모듈식 C++ 노드구조를 구현하여 장애물 회피, 최적 경로 이동을 실행하는 시뮬레이션 제시

## 2. 개발 환경
### 운영체제: wsl 혹은 VMware Workstation Pro환경의 Ubuntu 22.04LTS
### ROS버전: ROS2 Humble
### 개발언어: C++(ROS2)
### 시뮬레이션: 현재 TurtleBot3 Gazebo / 나중 

## 3. 현재 진행 상황
### 1) Docker 개발 환경 구축
### 2) Turtlesim 실행 후 토픽 구조 이해, 노득 간 통신 테스트
### 3) TurtleBot3 시뮬레이션 사용해보기
#### 이유: 실제 모델과 유샇나 구조를 가졌기 때문에 나중에 실제 자율주행 차량 플랫폼으로 사용하기 전 ROS2 Navigation 구조 연습이 필요했기 때문, 최종 시뮬레이션은 F1TENTH 환경으로 진행할 예정
### 4) ROS2 패키지 생성 및 노드 작성
#### package.xml & CMakeLists.txt 파일 작성
#### draw_square_node, wall_avoider_node, stop_node 구현

## 4. 실행 방법

### 2) 노드 실행
