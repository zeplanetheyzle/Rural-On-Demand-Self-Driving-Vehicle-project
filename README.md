# Rural-On-Demand-Self-Driving-Vehicle-project

## 1. 프로젝트 개요 및 동
### 문제 정의: 농촌 지역의 제한된 교통서비스로 인한 주민들의 이동권 보장
### 문제 해결책 제시: 농촌 주민들의 이동권 보장 및 삶의질 향상시키기 위한 자율주행차량 호출 서비
### 목표: ROS2 C++를 기반으로 모듈식 노드를 구현하여 시뮬레이션 제시

## 2. 개발 환경
### 운영체제: VMware Workstation Pro환경의 Ubuntu 22.04LTS 혹은 WSL
### ROS버전: ROS2 Humble
### 개발언어: C++(ROS2)

## 3. 현재 진행 상황
### docker를 설치하고 이 환경에서 my_turtle_controller_cpp(거북이 이동 코드), package.xml(패키지 정의 및 선언서)과 CMakeLists.txt(빌드 스크립트) 작성, colcon build까지 진행했어. 근데 기존 터미널을 닫고 다시 시작하면
### T시뮬레이션은 아니지만 거북이가 벽 만나면 돌아가는 것으로 만들었는데
### 핵심 기능을 수행할 C++노드 구조에 대한 계획 수

## 4. 실행 방법
### 1) ros2_ws에서 실행
### 2) 노드 실행
### 3) 시연(사각형 그리기, 비상 정지 시연, 벽 회피 시연)
